# Personal Training Platform (Live Code Editor)

A custom, browser-based Live Code Editor built from scratch using Vanilla HTML, CSS, and JavaScript. This platform serves as a personal training sandbox to practice web development, DOM Manipulation, and Event Handling in real-time.

## Features
- **Live Preview**: Instantly renders HTML, CSS, and JavaScript as you type, without needing to refresh the page.
- **Split Screen Layout**: Code editor on the left, live output `iframe` on the right.
- **Dark Mode UI**: Professional, hacker-style dark theme inspired by code editors like VS Code.
- **No Dependencies**: Built entirely with pure Vanilla JS and Flexbox.

## Tech Stack
- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript (Event Listeners, DOM Manipulation, Template Literals)

## How to Use
1. Clone this repository or download the files.
2. Open `index.html` in your favorite browser.
3. Write HTML, CSS, and JS in the respective text areas. The output will automatically update in the right panel!
